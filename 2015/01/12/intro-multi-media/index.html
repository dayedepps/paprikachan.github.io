Audio Basics

Analog to Digital Conversion

A simple analog-to-digital converter:

Analog Signal -> Sampling -> Quantization -> Digitized Codewords

analog signal -> sampling -> fitting curve (noise will be reduced)

Sampling

Sampling frequency is the number of samples per second, then the amplitude is held to allow quantization.

For telephone: 8000 samples/sec.

CD: 44100 samples/sec.

DVD: 48000 samples/sec.

Nyquist-Shannon Sampling Theorem

For no loss of information, sampling freq must >= 2 * max(signal freq), this is the highest frequency that can be accurately represented.

eg: human hearing < ~20KHz, CD sample rate: 44,100 samples/sec achieve Nyquist Shannon Theorem.

Quantization

sampled analog signal needs to be quantized or digitized. So, two questions need to solve: how many discrete digital values? what analog value does each digital value correspond to?

Simplest quantization: linear. 8-bit linear, 16-bit linear.

Aliasing Effects

For those lost higher frequencies, we add noise to sampled data at lower frequecies.

Alias frequency = abs(signal freq. - closest integer multiple of sampling freq.)

folding freq: the frequency above which aliasing occurs. calculated by halfing the sampling rate.

Example: 7KHz input signal, 8KHz sampling: 1KHz alias; folding frequency is 4KHz.
17KHz input signal, 8KHz sampling: 1KHz alias.
