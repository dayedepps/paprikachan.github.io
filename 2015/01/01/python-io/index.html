charset & encoding

阮一峰:ascii uninode and utf-8

文件类中文编码编译

# coding: utf-8

Example:

# coding:utf-8
s = u'你好'

编译出现错误:

UnicodeEncodeError: 'ascii' codec can't encode characters in position 0-15: ordinal not in range(128)

解决方法：设置python的默认编码为utf8格式。

sys.setdefaultencoding('utf8')

从MySQL获取中文数据

conn = MySQLdb.connect(user='root',charset='utf8')

JSON

使用json库。

>>> import json

Stream存为JSON格式，也就是dictionary

 >>> stream = '{"haha": "哈哈"}'
 >>> stream
'{"haha":"\xe5\x93\x88\xe5\x93\x88"}'
>>> js = json.loads(stream)  
>>> js
{u'haha': u'\u54c8\u54c8'}

json.load()        # 导入一条JSON数据
json.loads()    # 导入全部JSON数据

python数据格式转化为JSON: json.dumps()

>>>    print json.dumps(js)  
'{"haha": "\\u54c8\\u54c8"}'
>> print json.dumps(js, ensure_ascii=False)  
{"haha": "哈哈"}

python中json.dumps使用的坑以及字符编码

注意:在JSON.dumps()中key和value不能以混合普通字符串和unicode字符串的形式同时存在，否则会报错：

UnicodeDecodeError: 'ascii' codec can't decode byte 0xe6 in position 1: ordinal not in range(128)
