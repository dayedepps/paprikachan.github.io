module and package

  • Deal with module

    1. add the path when running the script

      import sys
sys.path.append(subdirectory)
import module_from_subdicrectory

    2. setting path in PYTHONPATH, so adding the path when shell start

  • Deal with package

    create __init__.py under all directories, to convert into a python package.

    from subdirectory import model_from_subdicrectory

    or

    from subdicrectory.module

Setting PYTHONPATH environment variable

In Linux/Unix, this script is usually located at ‘~/.bashrc’ or ‘~/.bash_profile’, use source to detect right the location.

source ~/.bashrc

Then, open the file in the editor, add

PYTHONPATH="${PYTHONPATH}:/path/to/some/cool/python/package/:/path/to/another/cool/python/package/"
export PYTHONPATH

Python Environment

virtualenv

conda manager

Conda is a package manager and environment manager.
Create an environment:

conda create --name envname biopython

Activate and deactivate environment

source activate envname
source deactivate

virtualenv Vs. conda

Create a virtual environment called “test”.
In conda, the package location should be ~/anaconda2/envs/test/
In virtualenv, the packages is located inside the folder “test”

os module

List files and directories

Current working directory name

os.getcwd()

list all files under a directory

from os import listdir
from os.path import isfile, join
onlyfiles = [f for f in listdir(mypath) if isfile(join(mypath, f))]

list all sub directories under a directory

[x[0] for x in os.walk(directory)]

Python中导入子文件夹中的模块

re module

正则表达式和python的re模块

Keras module

Keras高度模块化的神经网路库。基于TensorFlow或者Theano框架运行。快速实验。

yapf module

代码格式化工具，规范编码风格

tqdm module

|taqadum| 在长循环中添加一个进度提示信息，用户只需要封装任意的迭代器。快速，扩展性强的进度条工具库。

pyvim module

python 实现的 vim

snake module

snake用来取代vim的vimscript进行vim的插件编程

reverse-geocoder

用python实现的反向地理信息编码器(geocoder)，性能强，可离线。

pyxley module

使用Flask和React.js, 快速开发数据面板（dashboard）
相当于R的Shiny工具
不用谢html。css。可以有自己javascript来定制。

pupy module

远程管理工具（administration tool）开源，多平台。内置python解释器，可以从内存中加载python包，访问远程python对象。

tomorrow module

为python2.7中的异步代码提供了装饰器语法实现。

ibis module

Clouder labs推出。解决数据集规模问题，对用户提供的是单机python的体验。能与现有的python数据生态圈集成（pandas,scikit-learn,numpy）
