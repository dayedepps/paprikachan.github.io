Nan

Pandas will turn python.None into nan. numpy.isnan() and pd.isnull() can check if a dataframe value or series value is nan.

Pandas: handler missing data

Adding a new column to existing dataframe

df['newcoloum'] = df.Series

renaming columns in pandas

Rename all columns names:

df.columns = ['n1','n2','n3']

Rename part of columns names

df.rename(columns={'Leader': 'Commander'}, inplace=True)

Sort pandas by specified columns

df.sort(['col1'],ascending=0)
df.sort(['col1,col2'],ascending=[0,1])
df.sort(['col1'],ascending=True)

Select dataframe rows by row index

df.ix[], select the df rows based on its index, it can be any type.

indexs = df.index
for index in indexs:
    df.ix[index]

df.index[], select the df rows based on its actual integer indexs.

df[start_i,end_i], select the df rows based on its actual integer indexs, note that the end index is exclusive.

df[2:3]   ! the end index is exclusive

Iterate rows in Series

indexs = df.index
for index in indexs:
    value = df.ix[index]

Iterate rows in Dataframe

for index, row in df.iterrows():
    row['column_name']

Here, df.iterrows() returns a tuple pair (index, row) for each row line.
The type of row is pandas.Series.
