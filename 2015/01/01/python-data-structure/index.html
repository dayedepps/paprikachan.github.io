List, Set

List里面的元素可以用index调用，添加list.append(x)

Set里不能有相同元素，且里面的元素不可以用index调用，添加set.add(x), set的作用是把其他数据结构变为List

List和Set直接的相互转化：s = set(l); l = list(s)

Set操作

set表示无须集合，支持 x in set, len(set),和 for x in set。不支持 indexing, slicing, 或其它类序列（sequence-like）的操作。 

x = set([1,2,3])
y = set([3,4,5])

Difference x - y = set([1,2])

Union x | y = set([1,2,3,4,5])

Intersection x & y = set([3])

Symmetric Difference (XOR) x ^ y = set([1,2,4,5])

Superset x > y = Flase

Subset x < y = Flase

Sort

列表有自己的sort方法，对其列表进行原址排序。注意元组不可以使用这个方法因为元组不可修改。

如果需要一个sorted的新列表：

  • y = x[:]

  • sorted() 返回一个有序的副本，而且类型总是列表

Dictionary

Dictionary使用hash function构造，在找指定元素的时候很有效率。通过key查找对应的value效率为O(1)。

key和value可以为能够hash的任何类型： key::hashable value::hashable

当函数需要返回很多个值的时候，将返回值存为字典形式非常的方便，还可以保存错误信息。

dicts['Error'] = 'Type error.'
